# Quiz Algebra Lineare

Mini app statica per allenarsi con 120 quiz di algebra lineare.

## File necessari

- `index.html`
- `style.css`
- `script.js`
- `questions.js`

Caricali nella cartella principale del repository GitHub.

## Come pubblicarla con GitHub Pages

1. Vai nel repository.
2. Clicca **Settings**.
3. Vai su **Pages**.
4. In **Build and deployment**, scegli:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Clicca **Save**.
6. Dopo 1-2 minuti, GitHub mostrerà il link del sito.

## Funzioni

- risposta singola e multipla
- correzione immediata
- motivazione della risposta
- filtro per argomento
- modalità casuale
- modalità "solo errori"
